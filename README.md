# ❤️ Heart Health App

An Android application developed in Java that helps users monitor their heart health through cardiovascular risk assessments, vital signs tracking, and AI-assisted healthcare support.

This project was completed as part of a university software development module and showcases practical Android development using modern tools and technologies.

---

## ✨ Features

- 🔐 User registration and login
- ❤️ Heart health risk assessments
- 📊 Vital signs recording and monitoring
- 🤖 AI-powered healthcare chatbot
- 📁 Assessment history
- 👤 Patient profile management
- 🚑 Emergency healthcare assistance

---

## 🛠️ Technologies Used

- Java
- Android Studio
- XML
- Supabase
- OpenRouter API
- Git
- GitHub

## 💻 Key Skills Demonstrated

- Android application development using Java
- User interface design with XML
